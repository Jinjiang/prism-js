<prism-js>

A Polymer element for syntax highlighting with Prism.js

Usage

Specify language

<prism-js language="css"> p { color: blue; } </prism-js>

Specify language and theme

<prism-js language="css" theme="tomorrow"> p { color: blue; } </prism-js>

Point to a custom theme: <prism-js language="javascript" themepath="assets/monokai.css">

if (true) { console.log('foo'); }

Specify no language

<prism-js> p { color: blue; } </prism-js>

More examples

<prism-js language="css">

p { color: blue; }

<prism-js language="markup">

<button></button>

<prism-js language="css" theme="dark">

p { color: red; }

<prism-js language="css" theme="coy">

p { color: red; }

<prism-js language="css" theme="twilight">

p { color: red; }

<prism-js language="css" theme="tomorrow">

p { color: red; }

<prism-js language="css" theme="okaidia">

p { color: red; }

<prism-js language="javascript" theme="tomorrow">

if (true) { console.log('foo'); }

<p data-foo></p> <p data-foo ></p>
